This web application is created to manage dashboard files.
Here you can accomplish the following:

  • Create or log in to personal user interface

    • Please visit intrawebdev2/staff/sukharni/django/ to start. In case you don't have a password, register here: create new user

  • Upload your dashboards

    • After you logged in click on 'Upload dashboards' link and select title, status, type and resource of your dashboard, then select the file that you want to upload. Warning: you must fill ALL the fields in oder to proceed.

  • Download dashboards using site search system

    • Click 'Download dashboard' link. In the frame below choose one of the options:

    • Search by dashboard's name
      • This option allows for case-insensitive partial name matching
    • Select a dashboard from list
      • Simply select a dashboard from a scroll-down list

  • Leave comments and suggestions about site performance

    • By cliking 'Contact' you select an e-mail form. Your comments will be greatly appreciated!
